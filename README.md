# 🏏 IPL 2020 Data Analysis using Python

## 📖 Overview

This project presents an **Exploratory Data Analysis (EDA)** of the **Indian Premier League (IPL) 2020** dataset using Python. The analysis focuses on understanding team performance, toss decisions, match outcomes, venue statistics, and player achievements through data visualization and statistical analysis.

The project demonstrates essential Data Science skills including data cleaning, data exploration, visualization, and extracting meaningful insights from real-world sports data.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Explore the IPL 2020 dataset
- Analyze team and player performances
- Visualize important match statistics
- Generate insights using Python
